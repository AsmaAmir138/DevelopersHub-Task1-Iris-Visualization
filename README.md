# Task 1: Exploring and Visualizing the Iris Dataset

This repository contains my submission for **Task 1** of the AI/ML Engineering Internship at DevelopersHub Corporation.

## 📋 Objective
To load, inspect, and analyze the classic Iris dataset. The goal is to understand data trends, feature distributions, and relationships using visualization tools.

## 🛠️ Tech Stack & Libraries
- Python 3
- Pandas (Data loading and manipulation)
- Seaborn & Matplotlib (Data visualization)

## 📊 Key Insights
- **Pairplot Analysis:** Strong linear correlation observed between Petal Length and Petal Width.
- **Species Separation:** The *Setosa* species is easily distinguishable from *Versicolor* and *Virginica* based solely on petal measurements.
